LokaID API

Data Wilayah Indonesia untuk Developer

Versi 1.0.0

LokaID menyediakan API data wilayah administrasi Indonesia (provinsi, kota/kabupaten, kecamatan, dan desa) yang lengkap, terstruktur, dan mudah diintegrasikan. Ideal untuk validasi alamat, form, dashboard, atau aplikasi yang membutuhkan data lokasi Indonesia terkini.

Contoh: 
// GET daftar provinsi
fetch('https://lokaid.gilangpratama.id/states')
  .then(res => res.json())
  .then(data => console.log(data));

REST API

Akses data wilayah Indonesia melalui RESTful API dengan format JSON yang sederhana. Mendukung GET requests untuk semua level wilayah administratif.

LokaID API

Indonesian Region Data for Developers

Version 1.0.0

LokaID provides a comprehensive, structured, and easily integrated API for Indonesian administrative regions (provinces, cities/regencies, districts, and villages). Ideal for address validation, forms, dashboards, or applications that require up-to-date Indonesian location data.

Example: 
// GET list of provinces
fetch('https://lokaid.gilangpratama.id/states')
  .then(res => res.json())
  .then(data => console.log(data));

REST API

Access Indonesian region data via a RESTful API with simple JSON formatting. Supports GET requests for all administrative region levels.
